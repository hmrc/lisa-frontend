Register as a lifetime isa provider

You can use this service to register with HM Revenue and Customs (HMRC) as a lifetime ISA provider.

Before you start

You must:

  • be registered with the Financial Conduct Authority
  • be registered with HMRC as an ISA provider

You will need your:

  • corporation tax reference number
  • financial services register reference number
  • ISA provider reference number

If you do not have a Government Gateway ID, or need to re-register for an account, you can set one up as part of this service.
